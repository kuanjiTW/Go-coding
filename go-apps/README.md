
Introduction
============

This folder contains some simple Go applications.

- Terminal games
  + game-snake: adapted from [SnakeInGo](https://github.com/erap129/SnakeInGo)
  + game-tetris: taken from [go-tetris](https://github.com/cespare/go-tetris/)
- Web applictions
  + web-simple-app: adapted from examples [Go Wiki](https://go.dev/doc/articles/wiki/)
  + web-user-auth-app: a simple Web based login system
- Graphical user interface
  + gui-fyne-simple: the getting started example on [Fyne GitHub](https://github.com/fyne-io/fyne)
  + gui-fyne-demo: adapted from adapted from [fyne_demo](https://github.com/fyne-io/demo)
- AI applications
  + ai-genkit-samples: adapted from [Genki samples](https://github.com/firebase/genkit/tree/main/go/samples)
  + ai-welcome-note-generator: taken from [welcome-note-generator](https://github.com/vnaveen-mh/welcome-note-generator)
    with a [full artical](https://dev.to/naveen_v_5562871ba6455b7e/building-an-ai-powered-app-entirely-in-go-from-simple-prompt-to-smart-pipeline-237b)

See README.md files in each sub-folder for more details.
