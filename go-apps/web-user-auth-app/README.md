
Introduction
============

This program implements a simple Web based login system using libraries
[Gin](https://github.com/gin-gonic/gin),
[Gorm](https://github.com/go-gorm/gorm),
and a pure Go [sqlite3 driver](https://github.com/glebarez/sqlite).


How to Run
==========

1. Install dependencies.

   ```
   $ go mod tidy
   ```

2. Run the application.

   ```
   $ go run .
   ```

3. Visit the URL `http://127.0.0.1:8080` in a browser.
