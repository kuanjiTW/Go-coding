
Introduction
============

This program is the getting started example on
[Fyne GitHub](https://github.com/fyne-io/fyne).


How to Run
==========

1. Install dependencies.

   ```
   $ go mod tidy
   ```

2. Run the application.
   - Option 1: Run in desktop mode.

     ```
     $ go run main
     ```

   - Option 2: Run in mobile mode.

     ```
     $ go run -tags mobile main.go
     ```
